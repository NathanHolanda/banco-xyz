<h1>Running the project:</h1>
1. Firstly install dependencies:

```bash
npm install
```

<br>
2. To run Android:

```bash
npm run android
``` 
or to run iOS:
 ```bash
 npm run ios
```

<h1>Tests</h1>
1. To run tests:

```bash
npm run test
```
